Introduction:

The sysid package provides functions for constructing mathematical models of dynamic systems from measured input-output data. The package contains functions for data visualization, data preprocessing, parametric and non-paramteric model estimation, and model predictions and validation.

Overall Information Flow

Key Features:

  1. Data Acquisition:

    Data acquisition from mutliple sources - R workspace, table-formatted files, Excel spreadsheets and LibreOffice Documents

  2. Output Simulation:

    Generic sim to simulate outputs of Autoregressive (ARX and ARMAX), Output Error and Box Jenkins models

  3. Data Visualization and Preprocessing:
    • Detrending and resampling of data
    • Imputing Missing Data using linear approximation
  4. Non-parametric Estimation:
    • Estimating impulse and step response from data
    • Two routines (spa and etfe) for estimating the frequency response function.
  5. Parametric Estimation:

    Estimation of Autoregressive (ARX and ARMAX), Output Error and Box Jenkins models for SISO systems using prediction error minimization

  6. Model Validation:

    Model Validation using model predictions on the original data, new data and residual plots.

Features to be added in later versions:

  1. Estimation of transfer function and state space models
  2. Outlier Detection and Handling
  3. Generation routine for common input signal types - Random Gaussian, Random Binary, Psuedo-random Binary and Multi-sine
  4. Support for constrained modelling - options for fixed and free parameters
